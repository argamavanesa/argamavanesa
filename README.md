### ♿ TripWell — Inclusive Tourism Accessibility Platform *(Capstone Project)*
> **Tools:** Python, Google Colab, BiLSTM, NLP

AI-powered platform classifying Indonesian tourism destination reviews into accessibility categories (*Ramah Disabilitas* / *Akses Terbatas*) using Deep Learning and NLP techniques. Developed for Coding Camp 2026 Capstone.

**My Contributions (Data Analysis & Preprocessing):**
- Conducted full data wrangling and quality assessment on Kaggle tourism review dataset
- Performed text preprocessing: basic cleaning, slang normalization, and label encoding
- Conducted EDA and data visualization; preprocessing pipeline achieved **79% model accuracy**

**Key Insights:**
- Curug Malela identified as a significant negative accessibility outlier (52.5% negative reviews)
- Infrastructure-related terms (*jalan*, *akses*) dominate negative sentiment via c-TF-IDF analysis
- Review volume dropped sharply in mid-2022, attributable to post-pandemic mobility restrictions
